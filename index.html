To properly Jump to another part of the same page:

You need to place two items on each page:
    1. A basic link command pointing 
      to another section of the page.
    2. The point where the page will jump. 
    Here's the code:
    	Link: 
    	<a href=#SomeLabel>Click here</a>
    	Jumps To:
    	<a id="SomeLabel" name="SomeLabel">Another Section</a>
