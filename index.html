A JOURNEY OF BUILDING WITH PASSION

Maker • Builder • Thinker

His entire working life, Jess Heron's always been a craftsman of one sort or another. Now, he's applying the same time honored approaches to a new career in the tech industry.

These days, he creates mostly web apps, but he's also built houses, furniture, movies, videos, commercials, and he's written a screenplay.

Jess brings years of leadership competency creating products and meeting deadlines.His authentic personality, empathy, humor and intellect are an invaluable asset to any team.

As of January 2015 he's actively looking for the right fit for a fulltime web developer position. He can be reached through any of the contact links at this site.

Projects

Retreatly

Retreat.ly began as one of Jess's ideas for a Rails project at a Dev Bootcamp Hackathon.

As a portal for yoga instructors to handle and manage their retreats with minimal technical skill, the feature set includes:

• Account creation

• User Authentication

• CRUD operations

• Image uploading

• Reservation handling

• Payment system

Soapbox

Geo-specific, localized, multi-peer messaging web application.

Final project at Dev Bootcamp, Jess was responsible for appropriately modeling the geocaching distance methods in Ruby, some more generalized product input, as well as CSS and Front-End design.

Deployed at Heroku.

Candid.ly

Candid.ly is another brainchild of Jess Heron. Inspired by web apps Secret and Whisper, Candid.ly aims to build authentic friendships out of existing networks.

The project is built with Ruby on a Sinatra platform.

Digi4

AKA Digital Representation of Instatiated Competitive Placement of Consecutive Token Quadruplets.

This is a fun Tron themed Connect Four JavaScript implementation done as a group project during Dev Bootcamp.

Jess's work involved building the model and its various methods for board-state, turn-state & win-state in the group's MVC JavaScript app.
